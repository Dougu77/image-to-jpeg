# Image to jpeg

## Informações

- O programa faz converções em massa de imagens de diversos formatos diferentes para ```.jpeg```
- Formatos suportados: ```.png```, ```.jpg```, ```.jpeg```, ```.webp``` e ```.avif```
- Todas as instruções estão dentro do próprio programa.
